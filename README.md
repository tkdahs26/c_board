# c_board

자바로 여러가지 앱을 만들어서 언어의 영역을 넓히기 위해  visual studio로  c++의 구조와 문법을 이해하기 위해 프로젝트를 만들었습니다
c++은 게임이나 os개발에 쓰이고 struct는 자바의 필드 대신사용하고 vector,post와 같은 자료형,절차지향도 사용하는것도 알았고 자바와 다른 방식의 언어지만 자바와 문법 설계는 비슷하다는 것을 알았습니다.
DB연동,자료형만 알면 자바와같은 채팅같은 프로그램도 만들 수 있겠다 생각했습니다
